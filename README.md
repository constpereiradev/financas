## Financas

Base estrutural e organizacional dos projetos para formulários integrados com fluxo de aprovação

- composer install
- php artisan migrate
- php artisan storage:link 
- npm install
- npm rum watch
- php artisan db:seed


Principais Tecnologias:
- Laravel 10
- Vue
- Inertia
